# AI News Writing Workflow - My Contribution

## Overview
This repository documents my contribution to a collaborative news web application that used AI to rewrite RSS-based news content from a moderate perspective.

The goal of the application was to take content from RSS feeds and generate articles in a more neutral voice that emphasized truth, balance, and unbiased wording while avoiding strongly left-leaning or right-leaning language.

The main project repository is owned by my collaborator. This repository exists to present my role, technical contribution, and portfolio-ready documentation in a clear format.

## Project Summary
The application combined AI-assisted article writing with supporting website workflows for user engagement.

In addition to helping shape the AI writing behavior, I also worked on front-end changes, testing, and a newsletter signup workflow that allowed users to subscribe through a popup on the site.

## My Contribution
I contributed in four primary areas:

- Prompt design
- UI changes
- Testing
- Newsletter workflow implementation

## What I Worked On

### AI Prompt Design
- Helped design and refine prompts used to generate news articles in a moderate, neutral tone
- Improved prompt wording to better emphasize truth-focused and unbiased language
- Reviewed generated outputs and adjusted prompt instructions based on tone and consistency

### UI Changes
- Contributed front-end updates in React
- Improved parts of the user interface related to article display and user interaction
- Helped support a cleaner and more usable reading experience

### Testing
- Tested workflow behavior across the application
- Reviewed output quality, interface behavior, and site flow
- Helped identify issues related to usability, prompt behavior, and content handling

### Newsletter Signup Workflow
- Created a popup workflow that allowed visitors to sign up for the email newsletter
- Supported backend setup for storing user information with Supabase
- Connected Mailchimp for newsletter email workflow integration
- Helped connect user actions on the front end with backend data capture and email signup behavior
